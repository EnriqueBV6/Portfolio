📚 ¿Qué significan estos datos?

🔴 Masa Grasa ({{ "%.1f"|format(analysis.body_composition.body_fat_percentage) }}%)

Tu masa grasa es el {{ "%.1f"|format(analysis.body_composition.fat_mass) }}kg de tejido adiposo en tu cuerpo. Esto representa el {{ "%.1f"|format(analysis.body_composition.body_fat_percentage) }}% de tu peso corporal total.

Categoría: {{ analysis.body_composition.body_fat_category }}
{% if analysis.body_composition.body_fat_percentage <= 12 %} ✓ Excelente nivel de definición muscular. Ideal para competiciones o máxima visibilidad de musculatura. {% elif analysis.body_composition.body_fat_percentage <= 18 %} ✓ Muy buena composición. Equilibrio entre definición y sostenibilidad a largo plazo. {% elif analysis.body_composition.body_fat_percentage <= 25 %} ⚬ Composición normal. Adecuado para actividad física regular con buen rendimiento. {% elif analysis.body_composition.body_fat_percentage <= 32 %} △ Composición por encima del promedio. Considera aumentar actividad física para mejora. Riesgo cardiovascular moderado. {% else %} ✗ Composición con riesgo elevado. Se recomienda cambios en actividad y alimentación para mejorar salud cardiovascular. {% endif %}

💪 Masa Magra ({{ "%.1f"|format(analysis.body_composition.lean_body_mass) }}kg)

Tu masa magra incluye músculos, huesos, órganos y agua corporal. Es lo que queda cuando restas la grasa de tu peso total. Este valor es crítico para determinar tus necesidades de proteína.

Requerimiento de Proteína:
Se recomienda 1.6 a 2.2g de proteína por kg de masa magra
Tu rango: {{ "%.0f"|format(analysis.body_composition.lean_body_mass * 1.6) }} - {{ "%.0f"|format(analysis.body_composition.lean_body_mass * 2.2) }}g/día
Esto es más preciso que usar peso total, especialmente para personas con alta ó baja grasa corporal.

🏋️ Índice de Masa Libre (FFMI: {{ "%.1f"|format(analysis.body_composition.ffmi) }}/m²)

El Fat-Free Mass Index es similar al IMC, pero usa solo la masa magra. Es un mejor indicador de musculatura y composición que el IMC tradicional.

Categoría: {{ analysis.body_composition.ffmi_category }}
{% if analysis.body_composition.ffmi < 18 %} △ FFMI bajo - Masa muscular reducida para tu altura. Considera entrenamiento de fuerza. {% elif analysis.body_composition.ffmi < 20 %} ⚬ FFMI normal - Musculatura promedio adecuada para la población general. {% elif analysis.body_composition.ffmi < 22 %} ✓ FFMI atlético - Buena cantidad de masa muscular. Indica entrenamiento consistente. {% elif analysis.body_composition.ffmi < 24 %} ✓ FFMI muy atlético - Excelente desarrollo muscular. Posible uso de entrenamiento avanzado. {% else %} ✗ FFMI extremadamente alto - Raro sin intervenciones especiales. Verifica mediciones. {% endif %}

Interpretación:
FFMI < 18: Poco musculado | 18-20: Normal | 20-22: Atlético | >22: Muy musculado

📏 Índice Cintura-Cadera (WHR: {{ "%.2f"|format(analysis.body_composition.waist_hip_ratio) }})

El Waist-to-Hip Ratio mide la distribución de grasa y es un predictor importante de riesgo cardiovascular. Un valor alto indica más grasa acumulada en la zona abdominal.

Riesgo: {{ analysis.body_composition.whr_risk_level }}
{% if "bajo" in analysis.body_composition.whr_risk_level.lower() %} ✓ Excelente - Distribución de grasa saludable. Bajo riesgo cardiovascular. {% elif "moderado" in analysis.body_composition.whr_risk_level.lower() %} △ Moderado - Monitor tu peso abdominal. Actividad cardiovascular regular recomendada. {% else %} ✗ Elevado - Acumulación de grasa abdominal patogénica. Riesgo cardiovascular significativo. Cambios urgentes recomendados. {% endif %}

Referencia:
Hombres: <0.90 = Bajo | 0.90-0.99 = Moderado | >0.99 = Elevado
Mujeres: <0.80 = Bajo | 0.80-0.84 = Moderado | >0.84 = Elevado
